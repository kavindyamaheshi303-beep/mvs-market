📋 How to Update the Dashboard
Step 1: Fill in today's data below, OR click ⬆ Import Excel to auto-fill from your Excel file.
Step 2: Click 💾 Save data.json — this downloads a file called data.json to your computer.
Step 3: Go to your GitHub repo → click data.json → click the pencil ✏️ icon → paste the content → Commit. Done!
OR Step 3 (easier): In GitHub repo → "Add file" → "Upload files" → drag the downloaded data.json → Commit.
